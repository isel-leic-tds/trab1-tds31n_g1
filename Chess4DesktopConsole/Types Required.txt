Tipos:
   Player // WHITE | BLACK
   PieceType // King | Queen | Rook | Bishop | Knight | Pawn
   Piece(type,player)

   Column // a..h Coluna do tabuleiro
   Row    // 1..8 Linha do tabuleiro
   Square(column,row)  // Posição no tabuleiro
   Board  // O tabuleiro com as peças

   Move   // Regular(DONE) | Captura(DONE) | Promoção(DONE falta view) | Castling(DONE) | EnPassant

   Chess(board,moves)

onPressSquare:
    - ask model if it's the player turn
    - if not:
        - do nothing
    - if so:
        - see if there's already a square selected
        - if not:
            - see if the square is not empty
            - if not:
                - select square pressed
            - do nothing
        - if so:
            - ask model for Move object
            - ask model if there's possible to promotion
            - if not:
                - make the move on model
            - if so:
                - open window to ask for piece promotion
                - ask model to make new move with piece for promotion
                - make move on model
